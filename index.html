My Work

Game Price Tracker : Price comparison across different PSN stores over time, and currently through webscraping.

This project consists of a web application developed in Streamlit. Its goal is to provide real-time prices of games available in three different stores (ESP-USA-JAP) of PlayStation Network. Additionally, the application will display the price evolution of each game over time.

  • -- 1st Phase -- Main Development Phase (Now on development)
  • Developing a FastAPI-powered API to scrape real-time prices and creating an initial WebAPP using Streamlit.
  • -- 2nd Phase -- Resource and Code Optimization
  • Optimizing webscraping efficiency and enhancing real-time search capabilities.
  • -- Subsequent Phases --
  • Implementing new features and integrating with additional platforms.

Analysis of Visual Problems: Prevalence, Factors, and Economic Solutions

This project analyzes the prevalence and factors of myopia, as well as the competitiveness of prices in optical stores, using data from visual problems in the U.S. and in children. With datasets such as "Prevalence of Eye Diseases in the U.S." and "Myopia Study," and web scraping of optical stores in Spain, the goal is to provide practical solutions based on data analysis.

  • Prevalence of Visual Problems in the US (2017)
  • Factors in the Evolution of Myopia in Children
  • Economic Evaluation of Optical Solutions

Data Science Crew Challenge

This project is about helping tourists in Madrid by giving them info on the weather and city services. It uses temperature data and the City Council's API. With Python, it figures out and predicts temperatures and how many tourists might be around. The PythonAnywhere API gives predictions for temperature, the current temperature, and upcoming events for the next 10 days.

  • Data Collection
  • Data Preparation and Submission
  • Model Training
  • API Deployment
